# 💄 The Dolls — Luxury Makeup Website Design

A fully responsive, luxury-inspired **website design** for a beauty brand focused on inclusivity, self-expression, and high-end aesthetics.

## ✨ Overview

[![Live Demo](https://img.shields.io/badge/LIVE%20DEMO-LUXE%20Fashion-d87c4a?style=for-the-badge&logo=githubpages&logoColor=white)](https://dani1157.github.io/The-Dolls/)
**The Dolls** is a modern, visually rich **website design concept** created for:

* Trans women
* Non-binary individuals
* Anyone embracing bold, authentic beauty

The design combines elegant UI, immersive visuals, and interactive elements to deliver a premium, editorial-style experience.

---

## 🚀 Features

### 🛍️ Shopping Experience

* Dynamic product grid with categories (Lips, Eyes, Face, Sets)
* Product filtering & search functionality
* Quick add-to-cart interactions
* Interactive product modal views

### 🛒 Cart System

* Slide-in cart sidebar
* Real-time cart updates
* Item removal & total calculation
* Checkout-ready UI

### 🎨 UI / UX

* Luxury dark-themed design with gold accents
* Smooth animations & transitions
* Fully responsive layout (mobile + desktop)
* Interactive showcase panels

### 🌈 Inclusivity

* Trans-inclusive branding and messaging
* Dedicated “Trans Picks” section
* Pride-inspired visual accents and interactions

### 🔔 Extras

* Toast notifications
* Wishlist interactions
* Newsletter signup section
* Membership/pricing layout
* Testimonials carousel

---
